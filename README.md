# fintechWallet-mongodb-Ke
Simple fintech wallet using MongoDB

## 📌 Description
This project simulates a simple digital wallet system using MongoDB.

## 🚀 Features
- User creation
- Wallet management
- Add money
- Send money
- View balance
- View transactions
- Validation using MongoDB schema

## 🗂️ Collections
- users
- wallets
- transactions

## 🛠️ How to Run
1. Open MongoDB Shell
2. Run scripts from src folder step by step

## 📊 Example Operations
- Register user
- Add money
- Transfer money
- Check balance
